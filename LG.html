Multi-Dimensional Wheel Hub Measurement System

This project aims to improve efficiency, precision, and scalability in industrial inspections using a non-contact measurement system based on image processing. By combining CCD cameras, laser sensors, and advanced algorithms, the system provides reliable multi-feature detection of components, addressing key challenges in traditional quality inspection methods.

Background

Traditional inspection methods rely on manual tools like micrometers and gauges, which are labor-intensive, time-consuming, and prone to human error. These methods fail to meet modern industrial demands, especially for large-scale production, where precision, efficiency, and automation are critical. This system leverages non-contact image processing to overcome these challenges, offering significant improvements in accuracy, speed, and functionality.

Mechanical System

The mechanical system is a critical part of the measurement process, designed for precision and efficiency. It integrates the following:

  • CCD Camera Mount and Positioning:
    • The camera is mounted on a linear rail, enabling movement along X, Y, and Z axes.
    • Motorized adjustments ensure precise alignment for capturing images of varying object sizes.
  • Laser Sensor Integration:
    • Mounted on a rigid support structure to maintain stability during measurements.
    • Features vibration dampers to ensure accuracy in high-speed operations.
  • Fixture and Clamping Mechanism:
    • Includes pneumatically driven clamps with dual-action cylinders for stable and secure positioning.
    • Self-centering jaws ensure consistent alignment, while the rotary platform enables 360-degree measurements.
    • Constructed from lightweight, durable aluminum alloys for reliability in industrial environments.
  • Rotary Table:
    • Controlled by a high-precision servo motor to rotate components smoothly.
    • Works seamlessly with the pneumatic clamping system to measure all dimensions in a single cycle.

Image Processing Workflow

The system uses advanced image processing algorithms to calculate component dimensions. The workflow includes:

  1. Preprocessing: Converts raw images to grayscale, removes noise, and enhances edges.
  2. Contour Extraction: Identifies the boundaries of the object using morphological operations.
  3. Dimension Calculation: Determines critical parameters such as radius, area, and perimeter.
  4. Results Output: Outputs results in standard units and stores the data for further analysis.

System Features

  • Efficiency: Handles high-volume inspections in minimal time.
  • Precision: Achieves micrometer-level accuracy through advanced calibration.
  • Versatility: Measures dimensions and detects defects in various shapes and materials.
  • Real-Time Feedback: Provides instant defect detection and supports automated handling.

Applications

This system is applicable across multiple industries:

  • Automotive Manufacturing: Measuring wheel hub dimensions and bolt-hole alignment.
  • Electronics: Inspecting high-precision electronic components.
  • Aerospace: Analyzing irregular shapes like cam profiles and gear teeth.
  • Mass Production Lines: Real-time inspection and defect handling during large-scale manufacturing.
